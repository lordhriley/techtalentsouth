Analyzing Twitch Viewership Data in Python

// Quick Blurb
A quick project with python again since running Unity on my 4 y/o Mac was not ideal. This time, I made a tool that helps you analyze twitch user-video data. We use the Python requests library to make calls to the Twitch API and display the information with d3.js charts.
